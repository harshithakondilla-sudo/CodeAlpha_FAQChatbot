# CodeAlpha FAQ Chatbot

## Description
This project is an FAQ Chatbot developed using Python. It answers user questions based on a predefined FAQ dataset.

## Features
- User-friendly interface
- Answers frequently asked questions
- Fast response generation

## Technologies Used
- Python
- Streamlit
- NLTK

## Installation
```bash
pip install -r requirements.txt
```

## Run the Project
```bash
streamlit run app.py
```
